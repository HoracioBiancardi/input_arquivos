# Sistema de Ingestão de Arquivos

FastAPI + NiceGUI: upload de Excel/CSV/PDF, conversão para Parquet e envio para MinIO ou SQL Server,
conforme o contexto de negócio selecionado (ex.: "vendas"). Área administrativa (`/admin`) gerencia
contexts, usuários e audit log.

Ver `README.md` para instruções de instalação e execução.
